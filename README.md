# Malafrena 交互式图表系统

基于乌苏拉·K·勒古恩经典小说《Malafrena》创建的交互式可视化网页系统。

## 🌟 项目简介

本项目包含两个核心交互式网页：

1. **人物关系图** (`index.html`) - 展示小说中30个主要人物之间的复杂关系网络
2. **剧情走向图** (`plot.html`) - 展现小说七个章节的详细剧情发展

## 📊 功能特性

### 人物关系图
- **全面覆盖**: 包含小说中所有30个主要人物
- **关系可视化**: 使用不同颜色区分家庭、朋友、政治、恋爱等关系类型
- **交互式操作**: 点击人物查看详细信息和关系
- **原文展示**: 弹窗显示体现关系的原文段落及中文翻译
- **智能搜索**: 支持中英文人物名称搜索
- **动态布局**: 使用D3.js力导向图算法自动布局

### 剧情走向图
- **章节详解**: 七个章节的详细剧情解析
- **主题标签**: 每个章节都有相应的主题分类
- **关键事件**: 梳理各章节的重要事件节点
- **人物追踪**: 标注每个章节出现的主要人物
- **深度分析**: 包含主题分析和文学评论
- **交互展开**: 点击展开查看详细内容

## 🎨 设计特色

- **视觉美学**: 采用现代渐变色彩和精美排版
- **响应式设计**: 适配桌面和移动设备
- **流畅动画**: 平滑的过渡效果和交互反馈
- **用户友好**: 直观的操作界面和清晰的导航

## 🚀 使用方法

### 访问方式
1. 打开 `navigation.html` - 导航页面，可选择进入不同的图表
2. 或直接进入：
   - `index.html` - 人物关系图
   - `plot.html` - 剧情走向图

### 人物关系图操作
1. **浏览关系网**: 鼠标拖拽移动视图，滚轮缩放
2. **查看人物**: 点击任意人物节点查看详情
3. **搜索人物**: 使用左侧搜索框中英文搜索
4. **查看关系**: 在人物详情中点击关系查看原文证据
5. **控制视图**: 使用右上角按钮放大、缩小、重置或全屏

### 剧情走向图操作
1. **浏览章节**: 滚动页面查看七个章节
2. **展开详情**: 点击任意章节卡片展开详细内容
3. **快速导航**: 使用右下角导航按钮回到顶部或展开所有章节
4. **章节切换**: 点击"返回人物关系图"按钮在页面间切换

## 🛠️ 技术实现

- **前端框架**: 纯HTML/CSS/JavaScript
- **可视化库**: D3.js v7 - 力导向图
- **样式框架**: Google Fonts + Font Awesome
- **响应式设计**: CSS Grid + Flexbox
- **动画效果**: CSS Transitions + JavaScript

## 📚 数据来源

所有人物信息、关系数据和剧情内容均基于乌苏拉·K·勒古恩原著小说《Malafrena》进行提取和分析。

## 🎯 使用建议

1. **初次使用**: 建议从导航页面开始，先了解整体结构
2. **深入研究**: 使用人物关系图探索复杂的人物网络
3. **剧情理解**: 通过剧情走向图理解故事发展的脉络
4. **交叉参考**: 两个图表相互补充，提供全面的理解视角

## 📝 注意事项

- 所有人物名称使用英文原名，关系名称使用中文
- 人物介绍和描述采用中英文结合的方式
- 建议在桌面浏览器上获得最佳体验
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）
